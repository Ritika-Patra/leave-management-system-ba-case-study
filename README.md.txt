# Leave Management System – Business Analyst Case Study

## 📌 Problem Statement
The HR team of a 50-person startup manages leave requests through email and Excel, causing approval delays, balance errors, and poor visibility.

## 🎯 Objective
To automate leave requests, manager approvals, leave balance deduction, and reporting.

## 📂 Deliverables
- BRD
- FRD
- User Stories
- UAT Test Cases
- Process Flow
- Wireframes

## 🧩 Key Features
- Employee leave request portal
- Manager approval workflow
- Auto leave balance deduction
- Leave history
- HR dashboard
- Team overlap alerts

## 📈 Expected Outcome
- 70% HR effort reduction
- Same-day approvals
- Zero balance calculation errors
